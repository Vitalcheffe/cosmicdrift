## Overview

This is the **HarchCorp web platform** — a Next.js 16 application powering [harchcorp.com](https://www.harchcorp.com). It showcases our industrial sovereignty platform spanning AI infrastructure, cement, mining, water management, agriculture, energy, and sovereign intelligence across Africa.

## Key Verticals

| Vertical | Description |
|----------|-------------|
| 🧠 **HarchOS / AI Infrastructure** | Carbon-aware GPU orchestration with 1,798 GPUs across 5 Moroccan hubs |
| 🏭 **Cement & Construction** | Industrial cement operations across Morocco and The Gambia |
| ⛏️ **Mining & Minerals** | Phosphate, rare earth, and mineral processing technology |
| 💧 **Water Management** | AI-optimized desalination, water treatment, and distribution |
| 🌾 **Agricultural Technology** | Precision farming, vertical farming, and IoT crop monitoring |
| ⚡ **Energy** | Solar, wind, and green hydrogen infrastructure |
| 🔒 **Sovereign Intelligence** | Data sovereignty, submarine cable connectivity, and AI security |

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4 + shadcn/ui
- **Database**: Prisma ORM + PostgreSQL
- **Deployment**: Docker + Caddy
- **Charts**: Recharts + custom visualizations

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## About Harch Corp S.A.

Harch Corp S.A. is a Moroccan multi-sector industrial conglomerate and holding company headquartered in Casablanca, Morocco. We build Africa's industrial sovereignty through vertically integrated operations across 8 strategic verticals with a $2.4B+ investment pipeline spanning 5 countries.

Founded in 2024, Harch Corp operates across AI data centers, renewable energy, cement manufacturing, sovereign technology, strategic mining, precision agriculture, water infrastructure, and financial services — all designed to ensure Africa controls its own industrial destiny.

## HarchOS — Our Open-Source Platform

HarchOS is the operating system for sovereign AI infrastructure — built to turn Africa's renewable energy advantage into the world's greenest GPU compute platform.

| Repository | Description | Language |
|------------|-------------|----------|
| [**harchos-server**](https://github.com/HarchCorp/harchos-server) | Carbon-aware GPU orchestration platform — FastAPI backend with 30+ API endpoints | Python |
| [**harchos-sdk-python**](https://github.com/HarchCorp/harchos-sdk-python) | Python SDK with OpenAI-compatible inference, carbon tracking and async support | Python |
| [**harchos-sdk-js**](https://github.com/HarchCorp/harchos-sdk-js) | TypeScript/JavaScript SDK for sovereign AI infrastructure | TypeScript |
| [**harchos-cli**](https://github.com/HarchCorp/harchos-cli) | Command-line interface — multi-profile, structured output, CI/CD ready | Go |
| [**harchos-terraform-provider**](https://github.com/HarchCorp/harchos-terraform-provider) | Infrastructure-as-Code for sovereign GPU cloud | Go |
| [**harchos-grafana-plugins**](https://github.com/HarchCorp/harchos-grafana-plugins) | GPU monitoring, carbon tracking and sovereignty dashboards | TypeScript |
| [**harchos-examples**](https://github.com/HarchCorp/harchos-examples) | Starter templates for PyTorch, LLM inference and carbon-aware training | Python/TS |
| [**harch-corp**](https://github.com/HarchCorp/harch-corp) | Harch Corp web platform and industrial sovereignty tools | TypeScript |

## Key Concepts

- **Sovereign AI** — Data never leaves the region. Built for African data sovereignty requirements.
- **Carbon-Aware Computing** — Workloads automatically scheduled during low-carbon intensity windows.
- **OpenAI-Compatible** — Drop-in replacement for OpenAI API with `region="morocco"` and `sovereignty="strict"` defaults.
- **Pan-African Expansion** — Starting from Morocco's 5 GPU hubs, expanding across the continent.
